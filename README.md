# Terraform Hands On

## Commands

### Init

```bash
terraform init
```

### Plan

```bash
terraform plan
```

### Apply

```bash
terraform apply
```

### Destroy resource

```bash
terraform destroy -target MY_RESOURCE
```

### Plan destroy all resources

```bash
terraform plan -destroy
```

### Destroy all resources

```bash
terraform destroy
```

### Show

```bash
terraform show
```

### Refresh state

```bash
terraform refresh
```
